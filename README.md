# Code Ed-Hackathon
Build an OCR-based engine that scans, converts image text into downloadable text using machine learning tech.

## Tech Used
<ul>
  <li>Python</li>
  <li>Djnago framework</li>
  <li>Python-tesseract (optical character recognition tool)</li>
</ul>
